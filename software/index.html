Software I've worked on

SpiceGUI

SpiceGUI is program that aims to make circuit simulation on GNU/Linux operating systems easier with a modern and easy to use graphical user interface.

It requires Python 2, Gtk >= 3.10, ngspice and gnetlist. RPM packages are available.

Python GPLv3+
